# Key Input Demo (Educational)

⚠️ DISCLAIMER  
Questo progetto è a scopo esclusivamente educativo.

Il codice registra input **solo all’interno della finestra attiva** e **non intercetta la tastiera di sistema**.  
Non è un keylogger di sistema e non è progettato per attività invasive o illegali.

## Cosa fa
- Registra i tasti solo quando la finestra è in focus
- Mostra l’input in tempo reale
- Salva un log locale

## Requisiti
- Python 3
- tkinter (`sudo apt install python3-tk`)

## Esecuzione
```bash
python3 demo_keys.py
